Guía práctica – Semana 3 Día 2

Actividad:
Funciones simples en JavaScript: organizar decisiones y reutilizar lógica.

Repositorio:
Repositorio dispuesto por el docente.

Importante:
No debes trabajar directamente en main.

Cada estudiante debe crear una rama personal con el formato:

nombres_apellidos

Ejemplo:
maria_gomez
juan_perez
laura_torres_garcia

Evidencia:
La evidencia de esta actividad es únicamente el push de tu rama personal a GitHub.

No debes entregar nada en Moodle.
No debes enviar capturas.
No debes subir archivos adicionales.
No debes publicar enlaces por otro medio.

Comando nuevo del día:
npm init -y

Pasos obligatorios:

1. Clonar el repositorio

git clone URL_DEL_REPOSITORIO

2. Entrar a la carpeta del proyecto

cd NOMBRE_DEL_REPOSITORIO

3. Crear la rama personal

git checkout -b nombres_apellidos

4. Verificar la rama actual

git branch

La rama marcada con * debe ser tu rama personal, no main.

5. Crear package.json

npm init -y

6. Ejecutar el archivo de función simple

node funcion-saludo.js

7. Modificar el nombre de la comunidad en funcion-saludo.js

8. Ejecutar de nuevo

node funcion-saludo.js

9. Ejecutar la función de evaluación de información

node evaluar-informacion.js

10. Modificar los valores de entrada

Por ejemplo:
const resultado = evaluarInformacion(true, false)

11. Ejecutar de nuevo

node evaluar-informacion.js

12. Ejecutar la función de evaluación de propuesta

node evaluar-propuesta.js

13. Modificar los valores de entrada

Por ejemplo:
const resultado = evaluarPropuestaComunitaria(false, true)

14. Ejecutar de nuevo

node evaluar-propuesta.js

15. Completar los archivos:

semana3/conceptos-funciones.txt
semana3/bitacora-comando-dia2.txt
semana3/reflexion-procesos-comunitarios.txt

16. Revisar cambios

git status

17. Agregar cambios

git add .

18. Crear commit

git commit -m "Funciones simples en JavaScript semana 3 dia 2"

19. Publicar la rama

git push -u origin nombres_apellidos
