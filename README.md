# Programadores para la Paz – Semana 3 Día 2

## Actividad
Funciones simples en JavaScript: organizar decisiones y reutilizar lógica.

## Objetivo
Comprender desde cero qué es una función en JavaScript, cómo se usa para organizar instrucciones y cómo puede representar procesos claros de revisión, verificación y decisión.

## Enfoque ciudadano
Esta práctica se conecta con la organización de procesos comunitarios. Una comunidad necesita criterios claros para revisar información, evaluar propuestas y tomar decisiones responsables. En programación, una función permite organizar una lógica para usarla varias veces.

## Comando nuevo del día

npm init -y

Este comando crea un archivo package.json de forma automática.

En esta clase package.json solo se usará para reconocer que un proyecto puede tener información de configuración. No se instalarán paquetes.
